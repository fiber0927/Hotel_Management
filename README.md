# 酒店管理系统

> **版本**: v1.3.0
> **更新日期**: 2026-05-06
> **状态**: 开发版本，持续迭代优化中

## 项目简介

这是一个基于前后端分离架构的酒店管理系统，采用 Vue 3 + Spring Boot + MySQL 技术栈开发。系统分为顾客端和管理员端，支持房间浏览、在线预约、预订管理、客房服务等核心功能。

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | Vue 3 + Vite + Element Plus | 现代化前端框架，使用 Composition API |
| 后端 | Spring Boot 3.2 + MyBatis | Java 生态主流后端框架 |
| 数据库 | MySQL 9.0 | 关系型数据库 |
| 构建工具 | Maven (后端) / npm (前端) | 项目构建管理 |

## 快速启动

### 1. 数据库准备

```bash
# 登录 MySQL 并创建数据库
mysql -u root -p
CREATE DATABASE hotel_management CHARACTER SET utf8mb4;
EXIT;

# 执行初始化脚本
mysql -u root -p hotel_management < database/schema.sql
```

### 2. 后端启动

```bash
cd backend
# 使用 Maven 启动
mvn spring-boot:run
# 或运行 jar 包
java -jar target/hotel-management.jar
```

后端服务端口：**8080**
API 基础路径：`http://localhost:8080/api`

### 3. 前端启动

```bash
cd frontend
npm install
npm run dev
```

前端服务端口：**5175**
访问地址：`http://localhost:5175`

## 房间状态流转

```
可预订 → 已被预约 → 已入住 → 已占用(待确认) → 可预订
```

**状态说明**：
- **可预订**：空闲房间，可正常预订
- **已被预约**：已被预约，尚未入住
- **已入住**：正在使用中
- **已占用**：退房后待清洁，需管理员确认退房后恢复可用

**重要规则**：
- 顾客删除已确认的预订，不会释放房间
- 房间只有通过「入住 → 退房 → 管理员确认退房」流程才能恢复可用
