# 数据库设计说明

> **版本**: v1.0.0
> **更新日期**: 2026-05-06

## 一、主要表结构

### 1.1 admin 表（管理员表）

存储系统管理员信息。

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| username | VARCHAR(50) | NOT NULL, UNIQUE | 用户名 |
| password | VARCHAR(100) | NOT NULL | 密码（加密存储） |
| name | VARCHAR(50) | | 姓名 |
| created_at | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

### 1.2 customer 表（顾客表）

存储顾客账号信息，用于顾客端登录。

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| username | VARCHAR(50) | NOT NULL, UNIQUE | 用户名 |
| password | VARCHAR(100) | NOT NULL | 密码 |
| name | VARCHAR(50) | | 姓名 |
| phone | VARCHAR(20) | | 电话 |
| created_at | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

### 1.3 guest 表（客人表）

存储入住客人信息，与 customer 表不同，guest 是每次实际入住的客人记录。

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| name | VARCHAR(50) | NOT NULL | 客人姓名 |
| phone | VARCHAR(20) | | 联系电话 |
| created_at | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

### 1.4 room 表（房间表）

存储酒店房间信息。

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| room_number | VARCHAR(20) | NOT NULL, UNIQUE | 房间号（如 101, 201） |
| room_type | VARCHAR(20) | NOT NULL | 房间类型：standard/suite |
| floor | INT | | 楼层 |
| status | VARCHAR(20) | | 状态：available/booked/occupied/cleaning/maintenance |
| price | DECIMAL(10,2) | | 价格/晚 |
| created_at | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

### 1.5 booking 表（预约表）

存储客户预约信息。

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| guest_id | BIGINT | NOT NULL, FK | 关联 guest 表 |
| room_id | BIGINT | | 关联 room 表（预约的房间） |
| check_in_date | DATE | NOT NULL | 入住日期 |
| check_out_date | DATE | NOT NULL | 退房日期 |
| status | VARCHAR(20) | | 状态：pending/confirmed/cancelled/completed |
| special_requests | TEXT | | 特殊要求 |
| created_at | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

### 1.6 checkin_record 表（入住记录表）

存储实际入住记录。

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| booking_id | BIGINT | FK | 关联 booking 表 |
| guest_id | BIGINT | FK | 关联 guest 表 |
| room_id | BIGINT | FK | 关联 room 表 |
| check_in_time | DATETIME | | 入住时间 |
| check_out_time | DATETIME | | 退房时间 |
| status | VARCHAR(20) | | 状态：checked_in/checked_out |
| created_at | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

### 1.7 bill 表（账单表）

存储客户账单信息。

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| checkin_id | BIGINT | FK | 关联 checkin_record 表 |
| room_charge | DECIMAL(10,2) | | 房费 |
| other_charges | DECIMAL(10,2) | | 其他费用 |
| total_amount | DECIMAL(10,2) | | 总金额 |
| is_paid | TINYINT | | 是否已支付：0/1 |
| notes | TEXT | | 备注（如损坏赔偿） |
| created_at | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

### 1.8 housekeeping_task 表（客房服务表）

存储清洁和维修任务。

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | BIGINT | PK, AUTO_INCREMENT | 主键 |
| room_id | BIGINT | NOT NULL, FK | 关联 room 表 |
| task_type | VARCHAR(30) | NOT NULL | 任务类型：cleaning/repair |
| priority | VARCHAR(10) | | 优先级：high/normal/low |
| status | VARCHAR(20) | | 状态：pending/in_progress/completed |
| notes | TEXT | | 备注 |
| created_at | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| completed_at | TIMESTAMP | | 完成时间 |
| updated_at | TIMESTAMP | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

```
       ▼

       │                    │
       │                    │
       ▼                    ▼
┌─────────────┐      ┌─────────────┐
│checkin_record│      │    room     │
└──────┬──────┘      └──────┬──────┘
       │                    │
       │              ┌─────┴─────┐
       ▼              ▼           ▼

└─────────────┘  └────────────┘ └─────────────┘
```

## 三、初始化 SQL 脚本

数据库初始化脚本位于 `database/schema.sql`，主要包含：

### 3.1 创建数据库
```sql
CREATE DATABASE IF NOT EXISTS hotel_management CHARACTER SET utf8mb4;
USE hotel_management;
```

### 3.2 创建表结构
按依赖顺序创建表（先主表后从表）：
1. admin, customer（用户表）
2. room, guest（资源表）
3. booking, checkin_record（业务表）
4. bill, housekeeping_task（扩展表）

### 3.3 初始化数据

**管理员账号**:
```sql
INSERT INTO admin (username, password, name) VALUES ('admin', 'admin123', '系统管理员');
```

**顾客账号**:
```sql
INSERT INTO customer (username, password, name, phone)
VALUES ('test01', 'test123', '测试用户', '13800138000');
```

**初始房间**:
```sql
-- 标准间
INSERT INTO room (room_number, room_type, floor, status, price)
VALUES ('101', 'standard', 1, 'available', 299.00);

-- 套房
INSERT INTO room (room_number, room_type, floor, status, price)
VALUES ('201', 'suite', 2, 'available', 599.00);
```

## 四、索引说明

| 表名 | 索引字段 | 类型 | 说明 |
|------|----------|------|------|
| admin | username | UNIQUE | 快速登录查询 |
| customer | username | UNIQUE | 快速登录查询 |
| guest | phone | INDEX | 按电话查询客人 |
| booking | guest_id | INDEX | 查询某客人的预约 |
| booking | status | INDEX | 按状态筛选预约 |
| checkin_record | booking_id | INDEX | 查询预约对应的入住 |
| housekeeping_task | room_id | INDEX | 查询某房间的任务 |
| housekeeping_task | status | INDEX | 按状态筛选任务 |
